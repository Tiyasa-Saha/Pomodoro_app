# Pomodoro_app
Time yourself and focus better with this pomodoro timer

This is how it works:
1. Run the main.py file and the GUI opens up
2. Once you click start, the timer starts.
3. The timer starts off with your 1st work session of 25 mins and gives you a short break of 5 mins after.
4. The reset button resets the timer and you can start over.
5. After each work session completed, you get one 'check mark on the bottom'

The timer works in the following manner:
1. Session 1: Work - 25 mins
2. Session 2: Short Break - 5 mins
3. Session 3: Work - 25 mins
4. Session 4: Short Break - 5 mins
5. Session 5: Work - 25 mins
6. Session 6: Short Break - 5 mins
7. Session 7: Work - 25 mins
8. Session 8: Long Break - 20 mins
